# PDF 合并工具

这是一个用于合并PDF文件的Web应用程序。

## 功能特性
- 支持多个PDF文件合并
- 简单的Web界面操作
- 跨平台使用

## 使用方法

1. 打开 `docs/index.html` 文件
2. 点击"选择文件"按钮，选择要合并的PDF文件
3. 点击"合并"按钮
4. 下载合并后的PDF文件

## 项目结构

```
docs/
├── index.html        # 主界面
├── script.js         # 主要逻辑
├── styles.css        # 样式表
├── favicon.ico       # 网站图标
└── robots.txt        # 搜索引擎爬虫规则
```

## 依赖
