PDF Cross Merge Tool

Merge scanned document pages:

  • Front pages saved as front.pdf
  • Back pages saved as back.pdf
  • Reverse merge: Use when back pages are scanned in reverse order
  • Rotate pages: Use when back pages are upside down
  • Skip last page: Use when back file has one extra page

Runs locally in browser, files stay secure
